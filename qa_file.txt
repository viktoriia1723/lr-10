Питання: Як працює механізм обробки виключень try-except в Python?

Студент 2: Петрова
Відповідь: Механізм try-except в Python використовується для обробки виключень.
1. Код, який може викликати виключення, розміщується в блоці try
2. Якщо виникає виключення, виконання переходить до відповідного блоку except
3. Можна використовувати multiple except blocks для різних типів виключень
4. Блок finally виконується завжди, незалежно від того, чи виникло виключення

Питання: Які є основні структури даних в Python та їх особливості?

Студент 3: Сидорова
Відповідь: Основні структури даних в Python включають:
1. Списки (Lists) - упорядковані змінювані послідовності
2. Кортежі (Tuples) - упорядковані незмінні послідовності
3. Словники (Dictionaries) - невпорядковані колекції пар ключ-значення
4. Множини (Sets) - невпорядковані колекції унікальних елементів
5. Рядки (Strings) - незмінні послідовності символів
